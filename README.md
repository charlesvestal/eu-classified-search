Searches across multiple classified ad sites in the EU, focused on those that usually have musical instruments and audio equipment.

Usage: python search_ads.py "search term"

Supported sites:

Le Bon Coin (FR)
Marktplaats (NL)
Kleinanzeigen (DE)
Audiofanzine (FR)
Subito (IT)
Ricardo (CH)
Mercatinomusicale (IT)
Soudns Market (ES)
